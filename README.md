**项目简介**

这是一个基于Selenium+Pytest的售卖手机的电商网站自动化测试项目

涵盖了接口测试、数据库验证、UI自动化测试和压测

**技术栈**

我所使用的技术是Python,  Selenium,  Pytest,  requests

**快速开始**

1.先安装各种依赖，比如PHP扩展，composer，Apache，mysql，在Apache根目录下进入终端，建立项目的新文件夹

2. 从公司仓库克隆项目，切换到项目文件夹，输入命令composer create-project \
   --repository-url="https://username:password@172.168.23.157/realme/" \
   --stability=stable \
   realme/project-community-edition=8.0.0-p11 \
   ./Realme \
   --no-cache \
   --ignore-platform-reqs
   ————————————————
   3.导入测试数据，开始做接口测试。
